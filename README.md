# 杉並ホームバリュー

パークシティ杉並 セントラルタワーの3LDK参考価格と、東京23区の直近動向を確認する公開ダッシュボードです。

## 公開版の方針

- 建物全体について公開確認できる情報だけを掲載します。
- 個人の住所、部屋番号、購入額、改修費、融資・税務情報は掲載しません。
- 添付された契約書・登記資料などの原本や画像は、リポジトリへ保存しません。
- `ASK`（売出）、`CLOSE`（成約）、`INVENTORY`（在庫）、`ESTIMATE`（推定）を区別します。
- 公的な成約価格に週次系列はないため、「今週公表・確認できた最新材料」として月次・四半期データを読み直します。
- 新しい価格データがない日は、評価額を無理に動かしません。

## 主な機能

- 同一棟の公開参考事例から算出した3LDK価格レンジ
- 面積・階数・方角・室内状態によるブラウザ内の簡易補正
- 国交省の取引個票から独自集計した、東京23区・杉並区・主要区の3LDK㎡単価推移
- 2026年ナウキャスト、2027年予測、80%予測レンジ、実価格／2019年=100切替
- 東京23区の最新成約、城西地区の需給、募集賃料、金利環境
- 週次AIコメントと、次に確認すべき公表・イベント
- 出典、データ種別、基準期間、免責事項の明示

入力値は送信・保存されず、ブラウザ内だけで計算されます。

## 更新周期

- 営業日：金利、データ源の更新有無、鮮度
- 毎週月曜：東京23区の今週、AI市況コメント、次回確認日
- 毎月：東日本REINS、東京カンテイ、住宅ローン金利
- 四半期：国土交通省 不動産情報ライブラリの時系列再集計と予測誤差の検証

予測値はLLMではなく、公開数値を入力にした再現可能な統計式で算出します。LLMは根拠の要約と確認事項の整理だけに使い、新しい公表値がない日は予測を動かしません。
